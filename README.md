# StableGeo-BR
> Reliable and stable Brazilian geodata, from OpenStreetMaps and official resources.

Mapas GeoJSON das jurisdições do Brasil, e outros geodados brasileiros, confiáveis e estáveis, provenientes do OpenStreetMaps e de recursos oficiais, tais como IBGE, ANA e prefeituras.

Este repositório é mantido pela Digital-Guard em [`git.digital-guard.org/`**`stableGeo-BR`**](http://git.digital-guard.org/stableGeo-BR).
Os dados podem ser rastreados e reconstruídos a partir dos seus descritores em [`git.digital-guard.org/`**`preserv-BR`**](http://git.digital-guard.org/preserv-BR).

Os dados mais recentes deste repositório *git* estão sujeitos a correções, consulte o [versionamento](http://git.digital-guard.org/stableGeo-BR/releases) para baixar a versão desejada ou acompanhar o calendário da próxima homologação.

Licenças, metodologia e maiores de detalhes, consulte a documentação em `/docs`. Dados em `/data`.
